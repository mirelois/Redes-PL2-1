## Base Decisions

### Protocolo

### Métricas

 * *Jitter*
   * Variância da latência
 * *Bandwidth*
   * Provavelmente a mais complicada de gerir 
   * Menor *bandwidth* não impede necessariamente
 * Latência 
   * Tempo de viagem dos pacotes, entre envio/receção
 * Nº *hops* lógicos
   * Última métrica, pouco relevante, desempate
